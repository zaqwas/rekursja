Pseudokod funkcji rekurencyjnej

Dane: Liczba naturalna n.
Wynik: Pierszy oraz drugi element drugiego wiersza macierzy An.
Krok 1. Jeśli n = 0, to zwróć 1 oraz 0 i zakończ działanie funkcji.
Krok 2. Wywołaj funkcję rekurencyjnie dla ⌊n/2⌋, a obliczone wartości zapamiętaj w zmiennych f1 oraz f2.
Krok 3. Oblicz tmp := f1 · f1 + f2 · f2.
Krok 4. Oblicz f1 := f1 · f1 + 2 · f1 · f2.
Krok 5. Przypisz f2 := tmp.
Krok 6. Jeżeli n jest nieparzyste wykonaj:
oblicz tmp := f1 + f2
przypisz f2 := f1 oraz f1 := tmp
Krok 7. Zwróć wartość zmiennych f1 oraz f2 i zakończ działanie funkcji.

Pseudokod funkcji iteracyjnej

Dane: Liczba naturalna n.
Wynik: Pierszy oraz drugi element drugiego wiersza macierzy An.
Krok 1. Przyjmij a1 := 1 oraz a2 := 0.
Krok 2. Przyjmij f1 := 0 oraz f2 := 1.
Krok 3. Dopóki n > 0 wykonuj:
jeżeli n jest nieparzyste wykonaj:
oblicz tmp := f1 · a1 + f2 · a2
oblicz f1 := f1 · (a1 + a2) + f1 · f2
przypisz f2 := tmp
oblicz tmp := a1 · a1 + a2 · a2
oblicz a1 := a1 · a1 + 2 · a1 · a2
przypisz a2 := tmp
oblicz n := ⌊n/2⌋
Krok 4. Zwróć wartość zmiennych f1 oraz f2 i zakończ działanie funkcji.
