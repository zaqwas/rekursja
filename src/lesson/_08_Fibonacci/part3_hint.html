Wskazówka

W tej wskazówce postaramy się naszkicować jak może wyglądać iteracyjna funkcja „int fibonacci(n)”, która nie wykorzystuje żadnej tablicy. Na początku naszej funkcji może być sprawdzenie czy liczba n jest mała (np. zero albo jeden), wówczas możemy od razu podać wynik. Jeśli jednak liczba n jest większa, wówczas musimy do zmiennych a oraz b przypisać początkowe wyrazy ciągu. Następnie powinniśmy około n razy wykonać pętlę, która będzie wykonywać następujące operacje: najpierw należy wyliczyć kolejny wyraz ciągu i zapamiętać go w zmienne c (c := a1 + b), następnie wystarczy do zmiennej a przypisać wartość zmiennej b, a do zmiennej b przypisać wartość c. Po zakończeniu wykonywania pętli, wystarczy zwrócić zawartość zmiennej b.

Pseudokod algorytmu

Dane: Liczba naturalna n.
Wynik: N-ty wyraz ciągu Fibonacciego.
Krok 1. Jeżeli n ≤ 1, to zwróć wartość n i zakończ działanie funkcji.
Krok 2. Przypisz a := 1 oraz b := 1
Krok 3. Dopóki n > 2 wykonuj:
oblicz c := a + b
przypisz a := b oraz b := c
zmniejsz wartość zmiennej n
Krok 4. Zwróć wartość zmiennej b i zakończ działanie funkcji.
